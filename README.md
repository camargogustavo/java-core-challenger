 Sistema de Gestão Escolar

 # Sobre o Projeto
  - Este projeto é uma sistema de Gestão Escolas e foi feito utilizando a linguagem Java. Foi aplicado o conceito fundamentos da Programação Orientada Objeto trazendo uma simulação ao cenario real
    onde é necessário gerenciar professores, alunos, turma e semínario
  - O projeto foi desenvolvido como parte dos meus estudos na formação Maratona Java (DevDojo), focando em boas práticas de código e arquitetura limpa.

---

##  Funcionalidades

-  **Cadastro de Professores:** Associação de professores a seminários.
-  **Cadastro de Alunos:** Inclusão de alunos em seminários específicos.
-  **Gerenciamento de Seminários:** Vínculo entre local, título e participantes.
-  **Relatórios:** Impressão detalhada de todos os dados vinculados (quem está em qual seminário, qual professor ministra, etc.).

---

##  Tecnologias e Conceitos Utilizados

 **Java 17** (Linguagem base)
 **POO Completa:**
    * Classes e Objetos
    * Associação (1 para N, N para N)
    * Encapsulamento (Getters/Setters)
    * Construtores e Sobrecarga
    * Modificador `static` e `this`
* **IntelliJ IDEA** (IDE de desenvolvimento)
* **Git/GitHub** (Versionamento de código)

##  Como rodar o projeto

### Pré-requisitos
Antes de começar, você vai precisar ter instalado em sua máquina as seguintes ferramentas:
* [Git](https://git-scm.com)
* [JDK 17+](https://www.oracle.com/java/technologies/downloads/)
* Uma IDE de sua preferência (Recomendo o IntelliJ).

### Passo a passo

```bash
# 1. Clone este repositório
$ git clone [https://github.com/camargogustavo/seu-repositorio-aqui.git](https://github.com/camargogustavo/seu-repositorio-aqui.git)
# 2. Acesse a pasta do projeto no terminal/cmd
$ cd java-core-challenger

# 3. Abra o projeto no IntelliJ
# Certifique-se de marcar a pasta 'src' como Sources Root
# Caso não consiga rodar, dever ser a IDE que nao reconheceu que a pasta src é de java
  Se estiver no IntelliJ na aba lateral clique com botao direito em cima da pasta src, procure M"Make Directory As" e selocione a opcao Source Root(Azul)

# 4. Execute o arquivo Main (Geralmente em src/test/Main.java ou similar)
